📄 코팅기 통합 매뉴얼 (한글 + 문제대처 포함)
1. 🧩 도면 구조 설명 (Machine Layout)
✅ 주요 구성
[입력] → [저장 호퍼] → [진동 트레이] → [계량실]
          ↓
     [배출밸브]
          ↓
     [코팅 챔버]
          ↓
     [로터 + 스핀디스크]
          ↓
       [배출]

✅ 구성 요소 설명
① Input Hopper (입력 호퍼)

원료 저장
막힘 발생 시 공급 중단


② Vibrating Trough (진동 트레이)

거친 투입 / 미세 투입 분리
속도 조절 중요 (정확도 영향)


③ Weighing Chamber (계량실)

Load Cell로 무게 측정
± 오차 여부 관리 핵심


④ Release Valve (배출 밸브)

계량 완료 후 챔버로 이동
공압 제어


⑤ Coating Chamber (코팅 챔버)

실제 코팅 작업 공간


⑥ Rotor (회전기)

원료 혼합
코팅 균일도 핵심 요소


⑦ Spinning Disc (스핀 디스크)

코팅액 분사 담당
분사량 = 품질 핵심


⑧ Pump + Flowmeter

액 공급 제어
유량 정확도 중요


2. ⚡ PLC 입출력 정리 (현장용)
✅ 입력 (INPUT)

항목설명START 버튼운전 시작STOP 버튼운전 정지도어 센서문 닫힘 확인탱크 레벨코팅액 여부원료 감지원료 공급 여부Load Cell무게 측정온도/과부하모터 상태

✅ 출력 (OUTPUT)
항목설명로터 모터회전 작동진동 트레이원료 공급펌프코팅액 공급배출 밸브제품 이동도어 개폐챔버 개방알람 램프경고 표시

✅ 핵심 설명
👉 PLC는
“조건 맞으면 자동으로 순서 실행”
예:
원료 있음 → 계량 → 코팅 → 배출


3. 🚨 알람 코드 전체 해석
✅ 주요 알람
🔴 1. No Seed (원료 없음)

원인: 호퍼 비었거나 막힘
해결: 원료 추가 / 막힘 제거


🔴 2. No Coating (코팅액 없음)

원인: 펌프 문제 / 액 부족
해결:

탱크 채움
펌프 체크
호스 공기 제거




🔴 3. Tank Empty

원인: 코팅액 소진
해결: 즉시 보충


🔴 4. Chamber Door Open

원인: 문 열림
해결: 완전 닫기


🔴 5. Release Valve Open

원인: 밸브 닫힘 이상
해결: 공압 / 밸브 점검


🔴 6. No Container

원인: 수거통 없음
해결: 배출통 위치 확인


🔴 7. Inspection Door Open

원인: 점검문 열림
해결: 닫기


🔴 8. Motor Overload

원인: 과부하 / 막힘
해결:

내부 확인
모터 점검




4. 🔧 트러블슈팅 (완전 정리)

✅ ① 원료 문제

증상: 공급 안됨

증상: 밸브 / 도어 작동 안함

압력 5~8bar 유지
필터 청소


✅ ④ 센서 문제

증상: 무게 오류
해결:

15분 예열
0점 조정
10kg 보정


✅ ⑤ 코팅 불량

해결:

로터 속도 조절
보정 재실행


✅ ⑥ 배출 불량

해결:

공압 확인
밸브 점검

🔥 한줄 결론
👉 이 장비는
“공급 + 공압 + 센서” 3개만 관리하면 거의 모든 문제 해결됨
1. 장비 개요
본 장비는 **Film Coating Machine (필름 코팅기)**로
종자 및 곡물에 균일하게 코팅을 수행하는 설비이다.
✅ 주요 기능

자동 계량 (Load Cell)
회전 혼합 (Rotor)
액상 코팅 (펌프 + 디스크)
자동 레시피 제어 [코팅기_최종 | PDF]

✅ 처리 범위

2kg ~ 15kg


2. 작동 원리

원료 → 저장 호퍼 투입
진동 트레이로 계량
코팅 챔버 이동
로터 회전 혼합
코팅액 분사
균일 코팅 후 배출 [코팅기_최종 | PDF]


3. 주요 사양

전압: 220V (3상)
소비전력: 2kW
공압: 5~8 bar
집진: 500 m³/h
무게: 약 400kg [코팅기_최종 | PDF]
