# KKK
📄 코팅기 통합 매뉴얼 (한글 + 문제대처 포함)

1. 장비 개요
본 장비는 **Film Coating Machine (필름 코팅기)**로
종자 및 곡물에 균일하게 코팅을 수행하는 설비이다.
✅ 주요 기능

자동 계량 (Load Cell)
회전 혼합 (Rotor)
액상 코팅 (펌프 + 디스크)
자동 레시피 제어 [코팅기_최종 | PDF]

✅ 처리 범위

2kg ~ 15kg


2. 작동 원리

원료 → 저장 호퍼 투입
진동 트레이로 계량
코팅 챔버 이동
로터 회전 혼합
코팅액 분사
균일 코팅 후 배출 [코팅기_최종 | PDF]


3. 주요 사양

전압: 220V (3상)
소비전력: 2kW
공압: 5~8 bar
집진: 500 m³/h
무게: 약 400kg [코팅기_최종 | PDF]


4. 안전
전기

접지 필수
규정 전압 사용

기계

보호커버 제거 금지
유지보수 시만 개방 [코팅기_최종 | PDF]


5. 설치 조건

수평 설치
볼트 고정
공압 연결 (5~8 bar)
집진 연결 (500 m³/h) [코팅기_최종 | PDF]


6. 운전 방법
✅ 시작 전 확인

원료 있음
코팅액 있음
공압 정상
도어 닫힘

✅ 운전

START → 자동 코팅 시작
STOP → 현재 배치 완료 후 정지 [코팅기_최종 | PDF]


7. 유지보수

30분 이상 정지 → 호스 제거
물로 세척
2개월마다 베어링 윤활 [코팅기_최종 | PDF]


🧨 8. 문제 대처 (트러블슈팅)

🔧 ① 원료 부족 (No Seed)
증상

알람 발생
공급 멈춤

해결

호퍼에 원료 투입
막힘 제거
진동 트레이 동작 확인

👉 핵심
공급 부족 또는 막힘 문제

🔧 ② 코팅액 문제 (No Coating)
증상

코팅 안됨
액 안 나옴

해결 (순서 중요)

탱크 내용 확인 ✅
펌프 ON 확인
호스 공기 제거
유량계 확인
펌프 방향 확인
물로 테스트

👉 장비 이상 시
제품 별도 회수 후 재작업 [코팅기_최종 | PDF]
👉 핵심
액 부족 / 펌프 / 호스 문제

🔧 ③ 공압 문제 (Air 문제)
증상

밸브 안 열림
배출 안됨
문 안 열림

해결

공압 공급 확인
압력 5~8 bar 유지
필터 물 제거
솔레노이드 밸브 확인
실린더 작동 확인

👉 핵심
공압 없으면 기계 전체 멈춤

🔧 ④ 센서 / 계량 문제 (Load Cell)
증상

무게 오류
자동 동작 이상

해결

전원 후 15분 대기
0점 확인
10kg 추로 보정
센서 고정 확인
배선 점검 [코팅기_최종 | PDF]

👉 핵심
Calibration 문제 또는 설치 불량

🔧 ⑤ 코팅 불량
원인

속도 문제
레시피 오류

해결

Rotor 속도 조정
보정(Calibration) 재실행


🔧 ⑥ 배출 안됨
원인

공압 / 밸브 문제

해결

공압 확인
밸브 수동 테스트 [코팅기_최종 | PDF]


🔧 ⑦ 펌프 문제
해결

스위치 확인
방향 확인
호스 교체
물 테스트 [코팅기_최종 | PDF]


🔧 ⑧ 로터 안돌음
해결

세척
브러쉬 청소
모터 확인 [코팅기_최종 | PDF]


🚨 9. 작업자 긴급 점검 (중요)
문제 발생 시 무조건 이 순서:
1. 원료 확인
2. 코팅액 확인
3. 공압 확인 (5~8 bar)
4. 센서/알람 확인


✅ 10. 핵심 요약
👉 고장 원인의 대부분:

원료 부족
코팅액 문제
공압 부족
센서 오류

👉 해결 핵심:
💡 “공급 문제 → 공압 → 센서 순서로 확인”
